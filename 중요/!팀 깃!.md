# 깃 푸쉬 잘 못 했을 때 돌리는 법 ㄱ
**!!팀원들이 풀 안 했을 때만 가능!!!**

git checkout develop                     -> 원하는 브랜치로 이동 (디벨롭 브랜치 코드)
git pull                                       -> 최신 상태 풀
git log --oneline                           -> 삭제할(젤 위) 커밋과 그 전(바로 밑) 해시 확인
git reset --hard ==818b5f7==                 -> 삭제할 거 밑 해시 복사해서 노랑 자리에 붙이기
git push --force origin develop        -> (디벨롭)브랜치에 강제 적용

git log --oneline -5                        -> # 로컬 확인
git log origin/develop --oneline -5    -> # 원격 확인
ㄴ> 여기서 삭제할 커밋 해시가 안 보이면 완료
