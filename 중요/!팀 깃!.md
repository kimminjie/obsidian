git stash                       ->임시저장
git checkout develop       ->현재 브랜치에서 → `develop` 브랜치로 이동
git pull origin develop     ->서버(origin)에 있는 develop 최신 코드를
                        내 로컬 develop 브랜치로 가져오는 것
                        (두 개를 동시에 수행 명령어/ git fetch origin develop + git merge origin/develop)
